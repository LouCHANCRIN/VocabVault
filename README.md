# VocabVault

The goal of this application is to help me learn a language like other applications (Babel, Duolinguo, ...)

The stack is as follow:

## Frontend

### Stack

- Vue 3
- vuetify
- axios
- pinia

### Launch

Requirement:
- install npm

Commands:
- npm install
- npm run serve
